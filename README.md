# Sistema de Gestión de Reservas - Software FJ

Proyecto desarrollado para el curso de Programación - UNAD.

## Descripción
Este sistema permite gestionar clientes, servicios y reservas para una empresa llamada Software FJ utilizando Programación Orientada a Objetos en Python.

El proyecto implementa:
- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
- Manejo de excepciones
- Logs de errores y eventos

## Archivos del proyecto
- cliente.py
- servicio.py
- servicios_especificos.py
- reserva.py
- errores.py
- logger.py
- main.py

## Funcionalidades
- Registro de clientes
- Gestión de servicios
- Reservas
- Confirmación y cancelación
- Validación de datos
- Registro de errores en logs

## Tecnologías utilizadas
- Python
- GitHub
- Programación Orientada a Objetos
